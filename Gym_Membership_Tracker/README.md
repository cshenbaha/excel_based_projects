**Gym Membership Analytics Dashboard**
**Project Overview**
This project is an Excel-based Gym Membership Tracker designed to manage member registrations, monitor subscription durations, track expiry dates, and automate status updates. The system helps improve renewal monitoring and operational efficiency through real-time tracking and analytics.
**Objectives**
•	Track member joining and expiry dates
•	Automatically calculate remaining membership days
•	Classify membership status (Active, Expiring Soon, Expired)
•	Monitor renewal trends
•	Improve administrative efficiency
**Key Features**
•	Automatic expiry date calculation using EDATE
•	Real-time days remaining calculation using TODAY()
•	Status automation (Active / Expiring Soon / Expired)
•	Visual dashboard for operational insights
**Business Impact**
•	Reduces manual tracking errors
•	Improves renewal monitoring
•	Enables proactive member follow-up
•	Supports data-driven decision making
**Tools Used**
•	Microsoft Excel
•	Pivot Tables
•	Conditional Formatting
•	Excel Formulas (EDATE, IF, COUNTIF, TODAY)
**Future Improvements**
•	Revenue tracking module
•	Renewal rate analysis
•	Member segmentation
•	Automated reminder system integration
